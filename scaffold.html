Scaffold

A system for reducing overwhelm and supporting task initiation.

Designing for Neurodivergence

Scaffold functions as a cognitive prosthetic—externalizing executive function in moments where internal capacity is unavailable.

Focus Mode

The interface strips away all elements except the active task directive and a prominent, rhythmic visual timer.

What is happening

Once a step is accepted, the UI transitions into a dark, distraction-free focus mode. The screen displays only the immediate action required and a countdown timer, removing all navigation, secondary options, and visual clutter.

Why it matters

Minimizing external stimuli during initiation prevents tangential distraction. The visual timer creates a structured, finite container for attention, reducing the overwhelming feeling that the work will last indefinitely.
